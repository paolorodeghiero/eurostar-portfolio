<task type="auto">
  <name>Task 1: Verify backend APIs</name>
  <files>N/A - verification only</files>
  <action>
Run comprehensive API tests:

1. Committee workflow:
```bash
# Get committee status
curl http://localhost:3001/api/projects/1/committee

# Start workflow (should succeed)
curl -X PATCH http://localhost:3001/api/projects/1/committee-state \
  -H "Content-Type: application/json" \
  -d '{"committeeState":"draft"}'

# Progress to presented
curl -X PATCH http://localhost:3001/api/projects/1/committee-state \
  -H "Content-Type: application/json" \
  -d '{"committeeState":"presented"}'

# Try invalid transition (should fail with 400)
curl -X PATCH http://localhost:3001/api/projects/1/committee-state \
  -H "Content-Type: application/json" \
  -d '{"committeeState":"approved"}'
```

2. File upload:
```bash
# Upload a test PDF
curl -X POST http://localhost:3001/api/projects/1/business-case \
  -F "file=@test.pdf"

# Download the file
curl http://localhost:3001/api/projects/1/business-case/download -o downloaded.pdf

# Delete the file
curl -X DELETE http://localhost:3001/api/projects/1/business-case
```

3. Audit history:
```bash
# Get project history
curl http://localhost:3001/api/projects/1/history

# Verify entries show:
# - Correct user email (or dev-user)
# - Operation type (INSERT, UPDATE)
# - Field changes with old/new values
```

4. Alerts:
```bash
# Get active alerts
curl http://localhost:3001/api/alerts

# Check alert config
curl http://localhost:3001/api/alerts/config

# Update threshold
curl -X PUT http://localhost:3001/api/alerts/config/budget_limit \
  -H "Content-Type: application/json" \
  -d '{"budgetThresholdPercent":80}'
```

5. Budget triggers committee level:
```bash
# Update budget and verify committeeLevel is auto-set
curl -X PUT http://localhost:3001/api/projects/1/budget \
  -H "Content-Type: application/json" \
  -d '{"opexBudget":100000,"budgetCurrency":"EUR"}'

# Check project has committeeLevel
curl http://localhost:3001/api/projects/1
```
