<task type="auto">
  <name>Task 2: Create database schema with Drizzle ORM</name>
  <files>
    backend/drizzle.config.ts
    backend/src/db/schema.ts
    backend/src/db/index.ts
    docker-compose.yml
  </files>
  <action>
    Create database layer with all referential tables:

    1. Add to package.json: drizzle-orm, pg, @types/pg

    2. Create drizzle.config.ts:
       - Schema path: ./src/db/schema.ts
       - Out: ./drizzle (migrations folder)
       - Dialect: postgresql
       - dbCredentials from DATABASE_URL env var

    3. Create src/db/schema.ts with tables (using identity columns):

       **departments**: id (identity), name (varchar 255 unique), createdAt, updatedAt

       **teams**: id (identity), name (varchar 255), description (varchar 500 nullable),
                  departmentId (FK to departments, ON DELETE RESTRICT), createdAt, updatedAt

       **statuses**: id (identity), name (varchar 100 unique), color (varchar 7 for hex),
                     displayOrder (integer), createdAt, updatedAt

       **outcomes**: id (identity), name (varchar 255 unique), score1Example (text),
                     score2Example (text), score3Example (text), score4Example (text),
                     score5Example (text), createdAt, updatedAt

       **costCenters**: id (identity), code (varchar 50 unique), description (varchar 255),
                        createdAt, updatedAt

       **currencyRates**: id (identity), fromCurrency (varchar 3), toCurrency (varchar 3),
                          rate (numeric 10,6), validFrom (date), validTo (date nullable),
                          createdAt, updatedAt

       **committeeThresholds**: id (identity), minAmount (numeric 15,2), maxAmount (numeric 15,2 nullable),
                                 level (varchar 20 - 'mandatory', 'optional', 'not_necessary'),
                                 currency (varchar 3), createdAt, updatedAt

       **costTshirtThresholds**: id (identity), size (varchar 5 - XS, S, M, L, XL, XXL),
                                  maxAmount (numeric 15,2), currency (varchar 3),
                                  createdAt, updatedAt

       **competenceMonthPatterns**: id (identity), company (varchar 10 - THIF, EIL),
                                     pattern (varchar 500), description (varchar 255),
                                     createdAt, updatedAt

    4. Create src/db/index.ts:
       - Pool with connectionString from DATABASE_URL
       - Export drizzle instance with schema
       - Pool config: max 20, idleTimeoutMillis 30000

    5. Create docker-compose.yml:
       - PostgreSQL 16 service on port 5432
       - Volume for data persistence
       - Default credentials matching .env.example
  </action>
  <verify>
    docker-compose up -d
    cd backend && npx drizzle-kit generate
    npx drizzle-kit push
    # Connect to database and verify tables exist:
    docker exec -it $(docker ps -qf "ancestor=postgres") psql -U postgres -d eurostar_portfolio -c "\dt"
  </verify>
  <done>All 9 referential tables created in PostgreSQL with correct columns and constraints</done>
</task>
