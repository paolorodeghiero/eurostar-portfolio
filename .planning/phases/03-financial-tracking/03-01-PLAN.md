Add budget_lines table with these columns:

- fiscalYear: integer not null
- createdAt, updatedAt timestamps

Add unique constraint on (company, costCenterId, lineValue, fiscalYear) to prevent duplicate imports.

Follow existing schema patterns:
- Use integer('id').primaryKey().generatedAlwaysAsIdentity()
- Use numeric with precision 15, scale 2 for money
- Use timestamp with defaultNow() for audit columns
