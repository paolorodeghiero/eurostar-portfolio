  <action>
Run shadcn CLI to add all required components for the project sidebar:

```bash
cd frontend
npx shadcn@latest add sheet tabs slider command popover tooltip alert-dialog dropdown-menu collapsible textarea --overwrite
```

This installs:
- **sheet**: Sidebar overlay (slides from right)
- **tabs**: Vertical tab navigation (5 sections)
- **slider**: 1-5 value scoring
- **command + popover**: Combobox pattern for team selection
- **tooltip**: Disabled button explanations
- **alert-dialog**: Delete confirmation, conflict resolution
- **dropdown-menu**: Three-dot menu actions
- **collapsible**: Value score card expand/collapse
- **textarea**: Justification text fields

The command component combined with popover creates the Combobox pattern per shadcn docs.

Note: Use --overwrite flag to ensure latest versions, as some components may partially exist.
  </action>

Install use-debounce library for stable debounced auto-save:

```bash
cd frontend
npm install use-debounce
```

This library provides:
- `useDebouncedCallback`: For debounced save function
- `useDebounce`: For debounced values

Prefer this over custom setTimeout implementation to avoid stale closure bugs (per RESEARCH.md anti-patterns).
