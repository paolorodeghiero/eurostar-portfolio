- Body: { effortSize }
- Return 404 if team not assigned to project

- Return 400 if trying to remove lead team (isLead=true)
- Return 404 if team not assigned

Create backend/src/routes/projects/project-values.ts:

**GET /api/projects/:projectId/values** - List value scores:
- Return all project_values with outcome details (name, examples)
- Include score and justification
- Order by outcome name

**PUT /api/projects/:projectId/values/:outcomeId** - Set/update score:
- Body: { score, justification? }
- score must be 1-5
- Upsert: create if not exists, update if exists
- Return the value score with outcome details

**DELETE /api/projects/:projectId/values/:outcomeId** - Remove score:
- Removes the score for this outcome
- Return 404 if score doesn't exist

Validation for score:
```typescript
if (score < 1 || score > 5 || !Number.isInteger(score)) {
  return reply.status(400).send({ error: 'Score must be integer 1-5' });
}
```

The upsert pattern:
```typescript
await db
  .insert(projectValues)
  .values({ projectId, outcomeId, score, justification })
  .onConflictDoUpdate({
    target: [projectValues.projectId, projectValues.outcomeId],
    set: { score, justification, updatedAt: new Date() }
  });
```
