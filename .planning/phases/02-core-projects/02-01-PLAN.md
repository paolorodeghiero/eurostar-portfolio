Add the following tables to the existing schema.ts file:

1. **project_id_counters** - For PRJ-YYYY-INC generation:
   - year: integer primary key
   - lastId: integer default 0

2. **projects** - Core project table:
   - id: integer primary key (generated identity)
   - projectId: varchar(20) unique not null (PRJ-YYYY-00001 format)
   - name: varchar(255) not null
   - statusId: integer FK to statuses
   - startDate: date
   - endDate: date
   - leadTeamId: integer FK to teams (not null)
   - projectManager: varchar(255)
   - isOwner: varchar(255)
   - sponsor: varchar(255)
   - isStopped: boolean default false
   - version: integer not null default 1 (for optimistic locking)
   - createdAt, updatedAt timestamps
   - createdBy, updatedBy: varchar(255) for audit

3. **project_teams** - Involved teams with T-shirt sizes:
   - id: integer primary key
   - projectId: integer FK to projects (on delete cascade)
   - teamId: integer FK to teams
   - effortSize: varchar(5) not null (XS/S/M/L/XL/XXL)
   - isLead: boolean default false
   - createdAt, updatedAt timestamps
   - Unique constraint on (projectId, teamId)

4. **project_values** - Value scores per outcome:
   - id: integer primary key
   - projectId: integer FK to projects (on delete cascade)
   - outcomeId: integer FK to outcomes
   - score: integer (1-5)
   - justification: text
   - createdAt, updatedAt timestamps
   - Unique constraint on (projectId, outcomeId)

5. **project_change_impact** - Change impact teams:
   - id: integer primary key
   - projectId: integer FK to projects (on delete cascade)
   - teamId: integer FK to teams
   - impactSize: varchar(5) not null (XS/S/M/L/XL/XXL)
   - createdAt, updatedAt timestamps
   - Unique constraint on (projectId, teamId)

<task type="auto">
  <name>Task 2: Create project ID generator</name>
  <files>backend/src/lib/project-id-generator.ts</files>
  <action>
Create a new file that exports generateProjectId function:

```typescript
import { eq, sql } from 'drizzle-orm';
import { projectIdCounters } from '../db/schema';
import type { NodePgDatabase } from 'drizzle-orm/node-postgres';

export async function generateProjectId(db: NodePgDatabase<any>): Promise<string> {
  const year = new Date().getFullYear();

  // Upsert: increment counter or insert with 1 if not exists
  const [result] = await db
    .insert(projectIdCounters)
    .values({ year, lastId: 1 })
    .onConflictDoUpdate({
      target: projectIdCounters.year,
      set: { lastId: sql`${projectIdCounters.lastId} + 1` },
    })
    .returning({ lastId: projectIdCounters.lastId });

  const paddedSeq = String(result.lastId).padStart(5, '0');
  return `PRJ-${year}-${paddedSeq}`;
}
```

This atomically increments the counter for the current year and returns PRJ-YYYY-00001 format.
